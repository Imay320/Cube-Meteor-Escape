# 🎮 Meteor Escape (Dodge Game)

A simple and fun arcade-style browser game built using **HTML, CSS, and JavaScript**.  
Dodge falling obstacles like meteors, lasers, and robots to survive as long as possible!

## 🎯 Features

- Smooth keyboard controls (Arrow Keys)
- Multiple obstacle types:
  - ☄️ Meteors (rotating)
  - ⚡ Laser beams
  - 🤖 Robot blocks
- Score system
- Game over screen with restart option
- Neon-style modern UI

---

## 🕹️ How to Play

- Use **← Left Arrow** and **→ Right Arrow** to move
- Avoid all falling objects
- Survive as long as possible to increase your score
- Press **R** to restart after game over
